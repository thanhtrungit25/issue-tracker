# Issue Tracker (Node, React)

## Features of this project
- Express REST APIs
- MongoDB
- Modularization and webpack
- Routing with React Router
- Forms
- React bootstrap
- Server rendering
- Pagination
- Search Bar
- Google sign-in
- Session

```
# Clone
$ git clone https://github.com/thanhtrungit25/issue-tracker.git

# Install depdendencies
$ npm install

# App configuration
Change config file from static/config.js

# Run server and client
$ npm run dev-all
```
